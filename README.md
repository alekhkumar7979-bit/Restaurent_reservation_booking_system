 MERN Restaurant Reservation & Booking System

Project Description
This is a full-stack MERN application for restaurant reservation and booking. Users can browse menu, reserve tables, and manage bookings.

---

Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: CSS / Tailwind 

 Features
- User-friendly UI
- Table reservation system
- Menu display
- Backend API integration
- Full-stack MERN architecture
